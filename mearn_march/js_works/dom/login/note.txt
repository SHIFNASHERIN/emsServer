Dom methods - used to acess elements from dom 
-----------

document.getElementsByTagName('tagname')
document.getElementById('id')
document.getElementsByClassName('className')
document.querySelector('tagname/#id/.className')  - single element
document.querySelectorAll('tagname/#id/.className')  - multiple elements
