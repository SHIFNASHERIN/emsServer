array method

1) sort -  arrange the given array - ascending or descending - no separate array
2) map -  create new array having same number of elements
3) filter - creating new  array satisfying a particular condition
4) push - to add new elements elements to a given array
5) Pop -  to remove last element from the given array
6) find - return the first element that satisfy the condition
7)reduce -  higest value, lowest value , sum , average
8)some - it return true or false if the array satisfy the given condition
